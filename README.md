# FastAPI Project

## Описание

Это пример приложения на [FastAPI](https://fastapi.tiangolo.com/). FastAPI — современный, быстрый (high-performance), веб-фреймворк для создания API с Python 3.7+ на основе стандартного Python-type hints.
